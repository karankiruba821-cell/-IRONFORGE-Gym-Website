# -IRONFORGE-Gym-Website
IRONFORGE Gym Website is a modern, responsive fitness website built using HTML, CSS, and JavaScript.

## 🚀 Project Overview

It is designed to represent a professional gym brand with a **premium dark UI**, smooth animations, and interactive elements.

This project showcases front-end development skills by creating a complete gym landing page with multiple sections and dynamic user interactions.

---

## ✨ Features

* 🔥 Modern and attractive gym landing page
* 📱 Fully responsive (mobile, tablet, desktop)
* 🍔 Mobile-friendly navigation with hamburger menu
* 🎯 Smooth scrolling navigation
* 🎬 Animated hero section
* 🏋️ Training programs showcase
* 📅 Weekly class schedule (tab-based switching)
* 👨‍🏫 Trainers section
* 💰 Membership pricing plans
* 💬 Testimonials section
* 📩 Contact form
* 👀 Scroll reveal animations
* 📌 Active navigation highlighting

---

## 🛠️ Tech Stack

* **HTML5** – Structure
* **CSS3** – Styling, layout, animations
* **JavaScript (Vanilla JS)** – Interactivity
